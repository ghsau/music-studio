## 即梦操作步骤

1. 打开 [jimeng.jianying.com](https://jimeng.jianying.com) 登录
2. 视频生成 → 选**会员最新模型** → **全能参考**模式 → 比例 **9:16**
3. **逐段做**：上传该段 `@图片1` + `@图片2` + `@音频1`，粘该段 prompt，时长选 15s（末段除外）
4. 每段跑 **≥4 版**，按下方标准选最稳一版
5. 下载到 `assets/mv/mv1.mp4` … `mvN.mp4`，<N> 段都下完

## 选 take 标准（每段 ≥4 版里挑）

| 维度 | 加分 | 减分（直接 reject） |
|---|---|---|
| 风格一致 | 全程风格锚质感统一 | 突然偏离关键帧画风 / 俗气滤镜 |
| 不见脸 | 全程剪影 / 背影 / 局部 | 出现完整人脸 / 五官崩坏 |
| 内容对 | 该段主体 / 场景对得上关键帧 | 段落内容错（生成了别段的主体）|
| 音画合拍 | 歌词关键句处有镜头变化 | 整段画面静止 / 切换错节奏 |
| 画面节奏 | 每镜头 3-4 秒有呼吸感 | 快剪 / 一镜到底 |
| 跨段一致 | <N> 段像同一部片 | <N> 段明显是 <N> 个不同风格 |

## 跑完之后

<N> 段 mp4 都下到 `assets/mv/` 后，回来跑 `/make-video <曲名>`（断点后阶段）—— 自动拼接（ffmpeg concat + map 干净音轨）+ 烧字幕 → `final-mv.mp4`。

**同时记录即梦失败模式**（一镜到底 / 风格漂移 / 跨段割裂 / 内容错 / 出脸 / 不跟节奏），跑完一起复盘。
