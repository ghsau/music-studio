# 发布平台规格

> 由青衫/抖叔/小汽读写。**规格被平台拒过 = 必须更新这里**。

## 已验证（被实际平台规则证实）

- ✅ **汽水音乐 · 专辑封面**：必须 ≥ **1440×1440** JPG/PNG，1:1（《迟信》《今潮》《一城风》连续三首实测被拒，已沉淀）

## 已知"会卡 1440"的图像生成工具（按下载分辨率）

| 工具 | 默认下载尺寸 | 是否会卡 1440 | 解决 |
|---|---|---|---|
| **Gemini**（web 版） | **1024×1024** | ✗ 卡 | 下载后必走 `tools/check-assets.py` 自动放大；或换工具 |
| **Midjourney** | 1024×1024（默认） / 2048+（upscale 后） | 取决于是否 upscale | 下载前点 upscale；MJ Pro 还支持 2x/4x 二次放大 |
| **即梦**（字节） | 可选高分辨率档（2K / 4K） | ✗ 不卡 | 直接选高分辨率档下载 |
| **Stable Diffusion 本地** | 用户自定义 | ✗ 不卡 | 设 1500+ 即可 |

**实战经验（2026-05-06 高爽确认根因）**：连续三首歌的 1024 卡点都是 Gemini web 版下载默认尺寸——Gemini 不直接给 ≥1440 选项，只能下载完后端兜底。`tools/check-assets.py` 的 lanczos 放大对水墨/水彩/淡彩国风调容忍度高，目测无明显失真。

## 完整规格表（以平台规则为准；本表仅作下限）

### 汽水音乐
| 资产 | 规格 | 备注 |
|---|---|---|
| 专辑封面 | ≥ **1440×1440** JPG/PNG，1:1 | 低于会被拒；建议 2048+ |
| 音频 | MP3 ≥ 320kbps 或 WAV/FLAC | Suno 基础版输出 ~180kbps，Pro 版可下 WAV |
| 标题 | ≤ 30 字 | 含副标题 |
| 简介 | ≤ 200 字 | 三段结构最稳 |
| 标签 | 3~5 个 | 必带 #国风 |

### 抖音
| 资产 | 规格 | 备注 |
|---|---|---|
| 视频 | 1080×1920 MP4 H.264 + AAC | 9:16 竖屏 |
| 视频时长 | 1s–60s（卡点）/ 1s–10min（长） | 卡点切片优先 60s 内 |
| 视频封面 | 9:16 推荐 1080×1920 JPG | 也可截视频一帧 |
| 单视频字幕单行 | ≤ 12 字 | 手机竖屏可读上限 |

## 自动化兜底

`tools/check-assets.py`：扫 `<项目>/assets/`，对低于规格的封面**自动 lanczos 放大到 2x**（备份原图为 `<name>-original.<ext>`）。在 `/done` 流程里被自动调用。

升级路径：lanczos 放大效果不满意时，可上 AI 超分（`brew install upscayl-cli` / `pip install realesrgan`）。封面图本来就是水墨/绘画风时，lanczos 一般够用。
