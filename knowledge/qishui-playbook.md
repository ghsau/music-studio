## 已验证

### ✅ 轨迹 A：T+1 节令爆点 → 后续衰减

- 适用：节令向题材（假期 / 纪念馆 / 特定历史时刻 / 特定日子）
- 模式：发布日 T → **次日 T+1 单日峰值**（汽水算法首轮推荐窗口）→ 后续衰减
- 关键机制：节令题材的"假期归途反思"窗口 + 算法初推共振
- 操作建议：节令向题材发布时机对位"假期最后 1 天 → 收尾后首工作日"，配合 T+1 推送窗口
- 实战案例（具体歌名）见 `notes/qishui-playbook-attribution.md`

## 实验中

- 🧪 **轨迹 B：T+4~7 算法二次推 + 视频带动 → 持续上升**（**依赖偶然视频爆款，不可主动复制**）
  - 模式：发布后 T+4~T+7 出现单一视频爆款 → 算法二次推 → 持续上升曲线
  - **关键警告**：抖音视频播放高时**先看视频播放/使用量比例**，比例极端（>200x）说明单一视频独占流量，**非画像/hook/题材的稳定有效性**
  - 操作建议：不可主动复制（依赖运气）；可观察发布后 T+4~7 是否出现单一视频爆款，出现则乘势推

- 🧪 **算法二次推回升日**（无视频带动版）：上架后第 5-7 天可能出现算法二次分发（无平台运营动作、无视频带动），回升幅度通常较小

- 🧪 **汽水自动卡片视频 ≠ 抖音切片投入**：仅靠汽水音乐自动生成的卡片视频、不单独出抖音切片（15s/30s）或完整 MV → 抖音侧近零曝光（视频播放 + 音乐使用量贴地）。汽水自动卡片不进抖音推荐池。**要抖音流量必须单独出切片 / MV**（抖叔切片脚本 + `/make-mv` 命令均已就位）

## 种子内容

### 上传流程骨架

> 具体 UI 步骤可能随版本变化，此处只列必填字段与策略。用户首次跑通流程后可补充截图说明到本文末。

1. 进入汽水音乐创作者中心
2. 上传音频文件（mp3，建议 320kbps）
3. 上传封面图（正方形 ≥ 1000x1000）
4. 填曲名 / 简介 / 标签
5. 选择分类
6. 设置发布时间（立即 / 定时）
7. 提交审核

### 标题策略
- ≤ 30 字（汽水建议字数）
- 优先放副歌核心意象 + 情感词，避免硬塞流量词
- 考虑搜索匹配：包含 1 个核心意象词便于被搜到
- 避免：标题党、emoji 过多、《》《》连用

### 简介策略
- ≤ 200 字
- 第一段：曲风 + 情绪定位（"一首暮春题材的国风慢歌……"）
- 第二段：创作背景或情感故事（≤ 80 字）
- 第三段：可加一行欢迎评论分享的话

### 标签策略
- 选 3~5 个，宁少勿滥
- 国风画像必带：#国风 或 #古风
- chinese-rap 画像换：#中文说唱
- earworm-pop 画像换：相应轻快/派对类标签
- 风格类：#国风民谣 / #国风电子 / #戏腔 / #中文说唱 等
- 情绪类：#治愈 / #思念 / #国风燃曲 / #中年觉悟 等
- 避免：泛泛标签如 #好听 #推荐

### 置顶评论模板

#### 情绪共鸣型
- 「<歌词中最戳的一句>」+ 一行个人化体会

#### 互动钩子型
- 一个简短问题，引发评论区接龙

### 发布时间建议
- 周五 21:00 - 23:00（周末高峰前夜）
- 周日 20:00 - 22:00（次周准备前的情绪时段）
- 避开重大节日首日（流量被大平台占据）

### 抖音同步操作
- 在抖音用同一封面 + 切片视频发布
- 抖音文案 ≠ 汽水简介，要更口语化
- 带 1~2 个相关话题（#国风音乐 #原创音乐）
- 抖音视频时长建议 15s 或 30s（按抖叔脚本）
