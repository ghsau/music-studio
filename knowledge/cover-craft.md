## 一、封面 vs 首帧：两个不同的东西

很多人把"封面"和"首帧"当一回事，其实它们作用不同、出现场景不同，要分开设计。

| | 封面 | 首帧 |
|---|---|---|
| 是什么 | 用户主动上传/选定的那张图，显示在个人主页九宫格、合集、推荐流卡片 | 视频播放的第 0 帧画面，推荐流自动滑到时第一眼看到的画面 |
| 出现场景 | 主页浏览、搜索结果、合集列表 | 抖音推荐流（最大流量入口） |
| 用户行为 | 用户在"逛"，有时间扫 | 用户在"刷"，0.5 秒内决定划走还是停 |
| 设计重点 | 系列化、可识别、信息可稍多 | 极致简洁、瞬间钩子、不能靠文字慢慢读 |

**核心认知**：抖音绝大部分流量来自推荐流，所以**首帧比封面更决定生死**。封面影响的是"已经对你感兴趣的人来逛主页时点不点"，首帧影响的是"从没听过你的人刷到时停不停"。主理人条均点击率 0%——首帧问题大概率比封面更严重。

**首帧时长**：剪映/抖音可把封面帧设为 0.5 秒或 1 秒。设太长，用户点进来要先等封面播完，反而劝退。设 0.5 秒。

## 二、0.5 秒钩子：人为什么会停下来

用户刷推荐流，单条视频判断时间 < 1 秒。在这 0.5 秒里能让人停下的只有四种力量，优先级从高到低：

1. **对比冲击**（最强、最稳）—— 画面在一片同质内容里"跳出来"。亮主体 + 暗背景，或暗主体 + 亮背景。详见第三节。
2. **情绪张力** —— 一张有表情张力的脸（惊讶/动情/凝视），或一个有强烈情绪的瞬间。人脑硬件级优先处理人脸。
3. **好奇缺口（curiosity gap）** —— 画面留一个"没说完的问题"，用户必须点进去才能解答。比如两个矛盾元素并置、一个动作的中间态。
4. **信息钩子** —— 文字钩子（痛点/承诺/反常识）。对音乐作品来说优先级最低，慎用，见第五节。

**铁律**：一张封面/首帧只打一个钩子，打满。同时塞四个 = 一个都不成立。

## 三、高 CTR 封面的视觉工程

### 3.1 对比是王（Contrast is King）

对比比"用什么颜色"更重要。一个亮主体配杂乱背景 = 失败。

- **明暗对比**：深背景配浅主体，或浅背景配深主体。永远别让主体融进背景。
- **避开抖音/汽水 UI 同色**：抖音界面是黑底 + 红色进度条/按钮，推荐流里大量视频偏暗。封面若也是暗沉的中蓝/灰/黑，会"沉"进信息流不被看见。要么足够亮（暖黄、橙、米白），要么有一块足够亮的视觉锚点。国风题材常用低饱和墨色——这正是 CTR 风险点：低饱和 = 容易沉。解法见 3.4。
- **互补色制造冲击**：黄/紫、红/青、蓝/橙，色轮对位，冲击力最强。

### 3.2 单焦点（One Focal Point）

用户 0.5 秒看不懂复杂构图。最有效的封面只有 1 个主体（最多 2 个），其余全是陪衬。

- 主体在画面中占比 **50%-70%**：太小看不清，太大显挤。
- 背景极简：纯色、渐变、简单纹理。复杂图案会跟主体抢视线。
- 呼应青衫现有规则"一图 1~2 个核心物"——这里把它升级成 CTR 要求：不只是"别堆砌"，是"主体必须够大够清晰够跳"。

### 3.3 人脸与情绪（音乐作品要审慎）

通用数据：带人脸封面比纯物体封面 CTR 高 25-30%；带"惊讶/兴奋/好奇"表情的脸比无表情高约 30%。但有两个关键 caveat：

- **陌生人脸不一定加分，甚至减分**。主理人不是有辨识度的头部创作者，一张陌生脸 + 无信息 = 用户不知道为什么要点。人脸要"指向内容承诺"，不能"人脸本身就是承诺"。
- **2026 趋势是真诚 > 浮夸**。约 73% 用户更偏好真实、自然的表情，反感剧场式夸张表演。表情必须和歌的情绪一致。
- **视线引导**：脸朝向哪，用户视线就跟到哪。让脸看向画面里的关键元素（曲名/核心物），别让脸看出画面外。

音乐作品封面用人脸的正确姿势：不是"放个网红脸求点击"，而是用一个"有情绪、有故事感的人物瞬间"传递歌的情绪——是氛围载体，不是颜值噱头。题材没有人物时，不要硬塞人脸，用 3.4 的物象钩子。

### 3.4 国风/慢歌题材的特殊解法（低饱和 = CTR 风险）

国风慢歌天然偏低饱和、墨色、留白——美，但在推荐流里容易"沉"。不能为了 CTR 把国风做成高饱和俗艳，那毁了画像气质。解法是**局部对比 + 单点高光**：

- 整体可以低饱和，但留一个**小面积高饱和/高亮的视觉锚点**：一盏灯、一点朱砂红、一轮月、一片暖光。眼睛会被这个点抓住。
- **明暗结构而非彩度**：靠深浅对比拉开层次（如墨色背景里一个浅色剪影），不靠艳色。
- **情绪瞬间**：一个有张力的动作中间态（回头、伸手、抬眼），制造好奇缺口，比静态空镜更抓人。
- 反例：满屏均匀的灰蓝烟雨、没有任何高光点的纯水墨远山——美则美矣，推荐流里直接划走。

## 四、封面文字的取舍

**结论：抖音推荐流首帧默认不要 AI 生成文字；要加字，后期手动加，且严格克制。**

- 何同学 219 个封面测试结论：**封面文字若不能提供标题之外的额外信息或吸引力，就是冗余，反而分散注意力**。标题已说清主题时，封面加字是减分项。
- 工作室防呆规则已铁律 NO TEXT（AI 不会拼对中文 + 易出英文 banner）。本文件不冲突：文字一律后期手动加。
- 什么时候值得后期加字：
  - 字能制造"好奇缺口"或"情绪钩子"，且是画面给不了的信息（如一句歌词里最扎心的半句、一个悬念短语）。
  - 3-5 字以内，粗体无衬线（或题材匹配的书法体），手机缩略图下仍清晰可读。
  - 放中央安全区，避开底部 15%（抖音时长badge/UI 遮挡）和四角。
- 什么时候坚决不加字：画面本身钩子已成立、字只是复述曲名、字会盖住主体。

## 五、音乐作品/MV 类封面的特殊打法

音乐封面跟知识/带货封面不同——它卖的是"情绪和氛围"，不是"信息和结果"。

- **氛围即钩子**：音乐封面的好奇缺口往往是"这是什么样的情绪"。用色彩、光影、人物状态传递一种"想进去待一会儿"的氛围（参考 Billie Eilish 首专封面：强烈个人风格 + 一个奇特氛围，不靠信息）。
- **个人辨识度优先于单图点击**：音乐人要的是"听众认得出这是高爽的歌"。封面应有可延续的视觉签名（固定色系/构图母题/字体），跨歌一致——这样第 5 首歌的封面能借到前 4 首积累的认知。
- **色彩是音乐封面第一情绪语言**：先定情绪 → 定主色调 → 再定构图。冷暖、明暗直接对应歌的情绪光谱（与 styles.md 画像的「情绪光谱」字段对齐）。
- **MV 首帧 ≠ 封面**：MV 是视频，首帧要选"最完整、最有张力、最清晰"的一帧——避开动态模糊帧、过曝/过暗帧。抖叔的 mv-craft 管视频，青衫供图时关键帧也要按 CTR 标准选，不只是叙事正确。

## 六、汽水音乐封面 vs 抖音封面

两个平台，两套规格，不能一图通用。

| | 汽水音乐作品封面 | 抖音视频封面/首帧 |
|---|---|---|
| 宽高比 | **1:1 正方形** | **9:16 竖屏** |
| 推荐尺寸 | 3000×3000px（底线 ≥1440×1440，被实拒过） | 竖屏全屏适配 |
| 安全区 | 中央区域，歌词在封面上方滚动、左下/右下有点赞评论分享/队列按钮——**主体别压到四角和上下** | 主体压中央 1:1 区，避开底部 15% UI |
| 设计重点 | 沉浸式居中展示，氛围 + 辨识度 | 推荐流 0.5 秒抓眼球，钩子 + 对比 |
| 文字 | 可有少量（曲名），后期加 | 默认无字，钩子靠画面 |

实操：青衫一首歌应分别出**两版**——1:1 汽水版（氛围/辨识度导向）+ 9:16 抖音版（钩子/对比导向），不是把 1:1 裁成 9:16 了事。

## 七、benchmark：高 CTR 封面手法拆解

外部研究 + 案例拆出的可借鉴具体手法：

1. **何同学 219 个封面 A/B 测试**（数码博主，系统性多变量测试）
   - 手法：用小程序做多变量 A/B，α=0.05 显著性检验，逻辑回归/决策树量化各封面元素贡献度。
   - 可借鉴结论：① 封面文字无额外信息就是冗余；② 新人加人脸效果有限（头部创作者才靠脸强化品牌）；③ 高对比色 + 清晰主体 + 简洁构图是稳定的视觉锚点；④ 爆款 AirDesk 封面 = 简洁主体 + 科技感光线单点高光。
   - 给工作室的动作：主理人是"新人",别迷信人脸；先把对比和单焦点做到位。

2. **YouTube 高 CTR 缩略图研究**（百万级视频 A/B 数据）
   - 手法共识：单焦点 + 强前后景对比（避开平台界面色）+ 有情绪但内容诚实的表情 + 3-5 字粗体文字 + 留一个好奇缺口。
   - 数据点：带脸 +25~30% CTR，带惊讶/兴奋/好奇表情 +30%；亮红封面比蓝封面 CTR 高约 23%（红触发紧迫感，蓝接近界面色被吃掉）。
   - 给工作室的动作：国风慢歌不能用红色俗套，但"避开界面同色 + 留单点高光"这条直接适用。

3. **音乐专辑封面案例**（Billie Eilish 首专 / 流媒体封面设计）
   - 手法：纯数字发行时，版面给独特视觉而非说明文字；用强烈个人风格 + 一个奇特氛围抓注意力，不靠信息钩子。
   - 给工作室的动作：音乐封面卖氛围,建立可延续的视觉签名。

4. **抖音 2026 算法视觉信号识别**
   - 算法已能识别封面/首帧的视觉元素并据此分发（例：白色餐具关联"轻食"高净值搜索场景）。
   - 给工作室的动作：首帧的画面元素本身是分发信号——国风题材首帧出现明确的国风视觉符号（月/灯笼/古建/水墨），有助算法关联到对的人群。

## 八、青衫做封面/首帧的自检清单（交付前必过）

- [ ] 这张图打的是哪一个钩子（对比/情绪/好奇缺口/信息）？只打一个，打满了吗？
- [ ] 在缩略图尺寸 + 推荐流暗背景下，它会"跳出来"还是"沉下去"？
- [ ] 有没有一个明确的单焦点？主体占比 50-70%？背景够简洁？
- [ ] 低饱和题材：有没有留一个高亮/高饱和的视觉锚点？
- [ ] 用了人脸的话：表情真诚且与歌情绪一致？视线指向关键元素？不是为颜值硬塞？
- [ ] 文字：默认无；若加，是否提供了曲名之外的钩子？3-5 字？避开底部 15% 和四角？
- [ ] 汽水 1:1 和抖音 9:16 是否各出一版，而不是裁切了事？
- [ ] 防呆规则（NO TEXT / NO BORDER / East Asian 锚定 / Gemini 擦水印）全过了吗？
