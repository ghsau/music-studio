# MV / 短视频创作手艺 🧪

> 抖叔（视频导演）做任何 MV / 视频前必读。这是**创作手艺 / 吸引力层**——hook 工程、剪辑节奏、情绪曲线、完播率工程。
>
> **区别于 `knowledge/video-pipeline.md`**：那个是**技术管线**（Suno SRT / 即梦生成 / ffmpeg 拼接 / 字幕烧入），讲"怎么把视频造出来"；本文件讲"造成什么样才有人看完"。两文件配套用，不重复。
> **区别于 `knowledge/douyin-hooks.md`**：那个是**歌词层 hook 句式**（墨九写词用的句式模板），本文件是**视频画面 / 剪辑层手艺**。视频开场的"钩子字幕"可直接调用 douyin-hooks.md 的句式，但画面怎么配、镜头怎么切是本文件的活。
>
> **个人验证归因**（哪条手艺在哪首 MV 验证什么数据）记录在 `notes/mv-craft-attribution.md`，不入 git。本文件只装通用手艺 + 规则。

## 0. 为什么建这个文件 —— 真实数据诊断

工作室主理人抖音账号近 3 个月真实后台数据：

- 播放量中位数 **30**，账号陷在冷启动出不去
- **2 秒跳出率 40.58%** —— 40% 的人 2 秒内划走
- **5 秒完播率 25%**，**条均播放时长仅 5.78 秒**
- 单条歌曲 MV（盼长大 15s 短视频）完播率仅 **9.56%**

诊断核心：**不是账号限流，是视频活不过前 5 秒**——前 2 秒钩子失败 + 中段视觉太平导致流失。完播率是抖音决定推不推的核心指标。本文件就是把"开场 / 剪辑 / 节奏靠拍脑袋"换成"有研究过的手艺"。

**这个文件要解决的两个病灶**：
1. 前 2 秒钩子失败 → §1 hook 工程
2. 中段视觉太平 → §2 剪辑节奏与视觉密度

## 实验中 🧪 —— 种子手艺

### §1 前 3 秒 Hook 工程 🧪

> 算法事实：抖音前 3 秒完播率权重已提至约 40%，含封面点击率 / 2 秒跳出率 / 3 秒留存率三项。70% 以上的用户在 3 秒内决定走还是留。对标 TikTok：3 秒留存 65-85% 的视频拿到 2.2 倍曝光，低于 60% 几乎拿不到推荐。
> 工作室现状 2 秒跳出 40% → 目标先压到 30% 以下。

#### 1.1 视觉钩子必须独立于声音成立（核心铁律）

抖音首刷常静音。**把视频前 3 秒做出来，静音播一遍**——如果不靠声音就传不出"美 / 好奇 / 情绪"，这个开场就是废的，重做。
- 音乐 MV 尤其危险：抖叔容易默认"歌一响就有氛围"。错。首刷用户听不到歌。
- 解法：前 3 秒必有**视觉事件**——一个动作、一次镜头推进、一个钩子字幕、一个画面反差。不能是慢淡入 + 静止空镜。

#### 1.2 六类开场钩子模式（音乐 MV 适用）

| 模式 | 做法 | 音乐 MV 用法 |
|---|---|---|
| 悬念 | 抛出未解的画面 / 字幕 | 字幕"这首歌写的是我没说出口的一句话"+ 人物背影 |
| 反差 | 形象 / 场景 / 情绪的强对照 | 沧桑画面配清亮人声；热闹场景配孤独字幕 |
| 冲突 | 矛盾画面或矛盾文案 | "最热闹的孤独" + 人群中静止的人 |
| 信息前置 | 把最强记忆点搬到第 0 秒 | 副歌器乐前奏 / 副歌核心句直接开场，不用主歌铺垫 |
| 钩子字幕 | 第 0 秒压一行强字幕 | 调用 `douyin-hooks.md` 句式，画面同步给视觉锚 |
| 视觉冲击 | 高密度 / 高反差画面打断滑动惯性 | 强色彩、强构图、运动镜头作第一帧 |

**禁用开场**（直接导致 2 秒跳出）：慢淡入、纯黑首帧、静止 LOGO、空镜配前奏、"大家好"式自我介绍。前奏太长就**砍掉前奏从副歌切入**。

#### 1.3 钩子要"具体"不要"泛"

泛钩子（"这是一首关于成长的歌"）几乎无效。具体钩子（"写给我女儿出生那天的歌"）留存显著更高。
- 规则：开场字幕里必须有一个**具体的人 / 事 / 时刻 / 数字**，不能只有抽象情绪词。
- 钩子要在 3 秒内能被看懂——看懂耗时 5 秒的钩子等于浪费 2-3 秒生死窗口（认知流畅原则）。

#### 1.4 0-1 秒规则（与抖叔现有准则一致）

第一字 hook = 字幕 + 视觉同步出现在第 0 秒。这条抖叔 persona 已有，本文件确认并强化：第一帧不是"开始播放"，是"钩子已经在屏幕上"。

### §2 剪辑节奏与视觉密度 🧪 —— 治中段流失

> 工作室病灶：条均时长 5.78s，说明就算钩子留住人，中段也留不住。中段流失 = 视觉太平 + 信息密度不够。

#### 2.1 镜头时长与切换频率

- **快节奏**（适合冷启动突围）：平均镜头 1-2 秒切一次。"在镜头被看清之前就切掉"——美式预告片手法，节奏感天然不会差。
- **慢节奏**：3-5 秒一切，只适合内容粘性已经很强的成熟账号。**工作室现状（中位播放 30）不具备慢节奏资格——默认走快节奏。**
- 抖叔原准则"一个核心画面贯穿数秒再切"针对路径 A 单画面生成的局限，**但不等于全片静止**：单画面内要靠运镜（推 / 拉 / 摇）、景别变化、字幕推进制造内部运动，让"一个画面"也有节奏。

#### 2.2 视觉密度 / 信息密度

中段留人 = 提高单位时间信息量。原来 10 秒换一个画面信息 → 改成 5 秒一个 → 用户有"被持续喂养"的感受。
- 每 15 秒一个独立信息点 / 视觉变化，让留存曲线不出现平段。
- **但有上限**：信息密度过高会造成认知负担。原则是"张弛有度"——强拍密、弱拍疏。

#### 2.3 卡点 = 卡音乐的"呼吸点"，不是死卡鼓点

- 卡点不是机械切在每个鼓点。找音乐的**呼吸点**：鼓点、转调、歌手情绪爆发瞬间。
- 强拍 → 动作爆发 / 快切 / 视觉冲击；弱拍 / 转折 → 特写 / 慢镜 / 场景变换。画面与节奏要"对话"。
- 卡点要克制。专门为卡而卡显得做作假。卡在情绪点上才有意义。
- BPM-卡点换算见 `douyin-hooks.md`「BPM 与卡点关系」表，本文件不重复。

#### 2.4 拖沓段用变速救

中段如果有对情绪推进贡献小的段落，宁可加速压缩，不要指望用户耐心。变速本身也是节奏手段（剪映变速卡点）。

### §3 情绪曲线 / 结构 🧪

> 短视频也讲起承转合：起（钩子定调）→ 承（氛围 + 维持张力）→ 转（高潮 / 反转）→ 合（升华 + 引发评论）。

#### 3.1 四段结构与时间锚点

**15-30s 音乐 MV 短版**（工作室主力）：
- 0-3s 起：钩子完整抛一遍，定情绪基调
- 3-8s 承：第一个小起伏（节奏点 / 画面切换），防止"看着没内容"
- 8-15s 转：副歌核心句 + 视觉高潮集中在此
- 14-15s / 末段 合：留白或反问，引发评论

**60-90s 完整版 MV**：
- 高潮 / 情绪转折放在中后段（参 60s 视频约 31-45s 区间）
- 钩子不是只用一次——中段 15s / 30s 处补"中段留存钩子"对抗注意力自然衰减

#### 3.2 选段 = 找"能离开整首歌独立成立"的段落

音乐 MV 特殊性：歌是 60-90s，短视频要短，必须选段。
- **不要无脑选副歌**——选那个"脱离整首编曲后仍然有张力 / 有悬念 / 有记忆点 / 有身份感"的段落。有时是副歌，有时是一句反常的主歌。
- 优先选**歌词密度大、句与句衔接紧、没有长停顿**的段落（爆款翻唱《心恋》验证逻辑）——长停顿 = 用户觉得"没内容输出"→ 划走。
- 选段诊断三问：① 有没有一段 15s 能独立成立？② 有没有一句词能当 caption / 梗？③ 有没有一种情绪能对位常见视频格式？三个都"否" → 这首歌或这一段不适合做短视频，换段。
- 抖叔原准则"选连续段落不拼贴"仍然成立，本节是在"选哪一段连续段落"上加判断标准。

### §4 字幕 / 卡点 / 音画关系 🧪

- **字幕节奏**：每秒 ≤ 5 字推进，给读的时间。副歌 hook 句字幕停留 1.5-2s。（与 `douyin-hooks.md`「字幕节奏建议」一致）
- **打字机 / 入场动效**：字幕逐字出 + 停顿（标点 0.8s、句号 1.2s）能拉停留、造呼吸感。
- **音画同步**：字幕、视觉切换、卡点三者落在同一个音乐节点上，观感才"对味"。错位会让视频显廉价。
- **静音兜底**：承接 §1.1——关键字幕要保证静音也能读懂叙事。
- 字幕技术参数（font-size / --karaoke / 提前 0.3s）见 `video-pipeline.md`，本文件不重复。

### §5 完播率工程 🧪

#### 5.1 算法逻辑

- 完播率是抖音流量分发**第一道门槛**——完播率低于行业均值，再多赞也拿不到加权推荐。
- 流量池逐级晋升（赛马式）：新视频先推 200-500 泛用户，初始池核心考核完播率 + 互动率，过线才进下一级池。
- 分段考核：前 3 秒跳出率要 < 45%，后段也要持续留人。

#### 5.2 视频长度与完播率

- 15s 以内完播率最高（业界数据约 92%），16-30s 次之（约 84%），31-60s 明显下滑（约 68%）。
- **15s 以内视频完播率比 15s 以上高约 96%**，转发 / 评论也成倍。
- 但"原始观看时长"也是信号：40s/60% 留存（看 24s）有时比 20s/70% 留存（看 14s）拿到更多曝光。
- **工作室结论**：账号在冷启动期 → **默认做 15-30s 短版**，先用高完播率冲出流量池。完整版 MV 作品质需要另说，但不是冷启动突围工具。这就是"为什么视频做短"。

#### 5.3 互动延长停留

- 结尾留白 / 反问引导评论；评论区本身也是停留场（用户想看别人怎么说 → 完播率受益）。
- 适度争议性观点能拉评论，但音乐 MV 慎用，别为流量破坏作品调性。

#### 5.4 诊断工具

- 看抖音后台**完播率跳失曲线**：哪个秒数跳出高就改那一段。不要凭感觉改开场。
- 归因要追源：视频播放高可能是单一视频偶然爆款，不代表手艺普遍有效（见 `douyin-hooks.md`「归因要追源」+ `memory/feedback_review_attribute_root_cause.md`）。

### §6 素材选择 🧪 —— 有人有戏的名场面 >> 空镜

> 2026-05-18《春已来》验证：路径 C 用通用航拍空镜（长城/黄河/云海航拍）拼的 MV，再快剪、再卡点也是"一般"——空镜没有人、没有戏、没有瞬间，观众找不到情绪落点。同一条 MV 里董存瑞就义开场一眼就"对"，因为它有人、有戏、有名场面。一个有人有戏的镜头 > 一堆空镜。

- **铁律**：宏大叙事 / 爱国 / 情绪向题材，主体画面**优先"有人、有戏、有名场面"的素材**，通用航拍空镜只作极少量串场、不作主体。
- 素材来源优先级：① 高口碑、年代较近的主旋律影视剧片段（HD、戏剧张力强、天然吸引人，主理人验证「这种通常很清楚吸引人」）；② 真实重大仪式 / 名场面（阅兵、航天发射、航母巡航、民众欢庆、夺冠升旗）；③ 通用空镜垫底，少用。
- 画面要落到"人"——"长城外春已来"配的不该是长城航拍，该是民众仰头看升旗、孩子的笑脸。
- 选段 / 选素材时先问：这个镜头里有没有人、有没有事在发生、是不是一个"瞬间"？三个都没有 → 它就是空镜，换掉或只做串场。

## Benchmark 拆解 🧪 —— 高完播作品逐个学

> 跑新 MV 前抖叔过一遍，挑可借鉴手法。验证后的归因写 `notes/mv-craft-attribution.md`。

### B1. 阎其儿翻唱《心恋》—— 学"选段无停顿 + 人设反差"
- 爆点：一个月涨粉 100w+，杨超越 / 赵露思都拿来当 BGM。
- 学到：① **选段歌词密度大、句句衔接、无长停顿**——从根上消除"没内容输出"的划走动机（直接对治工作室条均 5.78s 病灶）；② 形象（朝天辫 + 吉他弹唱）、嗓音、古灵精怪表情制造**反差感**——反差就是 §1.2 的开场钩子。
- 抖叔可借：选段先做"停顿审计"；路径 B 数字人 MV 要给歌手设计有记忆点的表情 / 形象反差。

### B2.《零距离的思念》—— 学"器乐记忆点前置 + 话题钩子"
- 爆点：抖音累计播放破 200 亿，投稿 1300w+。
- 学到：① 把最具记忆点的**器乐段落（抹弦贝斯）放在视频开头**，不是从人声主歌起 → §1.2 信息前置的范本；② 配一句强冲击话题标签当钩子（"这首 R&B 一上来就强吻"）；③ 设计**可模仿的二创玩法**（冰箱转场）放大传播。
- 抖叔可借：选段时优先看歌曲有没有强记忆点的器乐 hook 可前置；MV 留一个"别人能模仿的动作 / 转场"。

### B3. 周伦伦原创 —— 学"快转场 + 漫画感花字救平淡"
- 爆点：理解短视频调性后，1-2 月涨粉到 10w（此前 20w 使用量只涨 100 粉）。
- 学到：原创音乐人光有歌不够，**迅速转场 + 漫画感后期花字**让短视频"成熟精致"——这是直接治 §2「中段视觉太平」的手法。
- 抖叔可借：中段不要长镜空着，用转场 + 花字提升视觉密度；后期花字是低成本提密度手段。

### B4. 阎/泛抖音爆款共性（"研究 1000 条"结论）—— 学"开头设期待感"
- 学到：爆款都在开头用音乐 / 人物 / 视觉 / 文案建立"好看 / 好奇 / 感动 / 有新知"的**期待感**；BGM 必须契合情绪不是随便套热歌。
- 抖叔可借：每条 MV 开场自检——"这 3 秒让用户期待接下来会发生什么吗？"

### B5. TikTok 独立音乐人 POV 叙事钩子 —— 学"一句话框定身份"
- 案例：indie 歌手发"POV: 你在听一个 22 岁布鲁克林音乐人的未发布歌曲" → 两周 120w 播放、涨粉 42%。
- 学到：**POV / 第一人称叙事框**一句话就建立好奇和身份代入，零铺垫。比"我做了首新歌"强 23% 量级（具体痛点 / 身份开场 vs 泛开场）。
- 抖叔可借：开场字幕用 POV / 第一人称句式，把"高爽的一首新歌"换成"POV: 一个父亲写给女儿的歌"这类具体身份框。

### B6. TikTok 音乐推广通则 —— 学"过程感 / 真实感 > 精致度"
- 学到：TikTok 数据显示 15s 的"扒人声分层"过程片段，比 60s 精致 MV 表现更好；真实 / 脆弱感建立连接。60% 高表现音乐类内容用情绪钩子。
- 抖叔可借：路径选择上，不是只有精致生成画面才行；幕后 / 创作过程 / 真实故事片段也是有效格式，且成本低。

### Benchmark 拆解共性总结
1. 选段独立成立、无长停顿 > 无脑选副歌（B1）
2. 最强记忆点（器乐 / 副歌 / 反差）前置到第 0 秒（B2、B4）
3. 中段用转场 + 花字 + 快切提密度，不留平段（B3）
4. 开场建立"期待感 / 身份代入"，POV 句式好用（B4、B5）
5. 真实感 / 过程感是有效且低成本的格式（B6）
6. 留一个可被模仿的玩法放大二创传播（B2）

## 给抖叔的执行清单（每条 MV 自检）

- [ ] 前 3 秒静音播一遍，不靠声音能不能传出钩子？传不出 → 重做开场
- [ ] 第 0 秒有钩子字幕 + 视觉事件？（不是慢淡入 / 空镜 / LOGO）
- [ ] 钩子里有具体的人 / 事 / 时刻 / 数字？不是纯抽象情绪词
- [ ] 选段做过"停顿审计"——没有长停顿、句句衔接？
- [ ] 选段过"独立成立三问"？
