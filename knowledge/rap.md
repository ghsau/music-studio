## 一、基本概念：bar / 小节 / 押韵密度

写词和写 prompt 前先对齐术语，否则墨九阿声握手时各说各话。

| 术语 | 含义 |
|---|---|
| **Beat（拍）** | 节奏的最小单位。中文说唱基本都是 4/4 拍 |
| **Bar（小节）** | 4 个 beat 为一组。两层含义：① 音乐结构上 = 4 拍的伴奏单位；② 写词上 = "一行词"。rapper 说"写了多少 bar"指的是行数 |
| **Verse（主歌段）** | 由 8–16+ 个 bar 组成。**标准是 16 bar**（"write a sixteen"），工作室柔情向 rap 可短至 8–12 bar |
| **Hook / Chorus（副歌）** | 4–8 bar，记忆点所在，可 melodic singing |
| **押韵密度** | 固定 bar 数内韵脚出现的频率与密集程度。不是越高越好——密度服务于情绪，柔情向宜疏，快嘴向宜密 |
| **Flow** | 字句在 beat 上流动的方式 = 节奏划分 + 腔调（旋律）+ 咬字。换 beat、换嗓音仍能辨认出的那个东西 |

**写词时的 bar 意识**（墨九）：一行词 = 一个 bar。一个 bar 4 拍，常规填字 4–8 字（每拍 1–2 字）；double-time 段可塞到一拍 3 字以上。**每行字数不能忽长忽短**——句与句字数差太多 flow 会断。

## 二、Flow 体系

Flow 是写词和选 beat 共同的锚点。墨九按 flow 排字数和停顿，阿声按 flow 选 beat 风格。**先定 flow，再定字，再定 beat**——不是先写满词再硬塞进 beat。

### Flow 的三个维度
1. **节奏划分**：韵脚和重音落在 beat 的哪些点上（三连音 / 切分 / 反拍 / 掉拍）
2. **腔调（旋律性）**：rap 段是否带旋律走向（柔情向常带，快嘴向偏念白）
3. **咬字**：清晰 vs 模糊，密 vs 疏

### 主要 flow 类型

| Flow 类型 | 节奏特征 | 适配题材 / 情绪 | 工作室画像对位 |
|---|---|---|---|
| **Boom-bap flow** | 老学校规整律动，鼓点"嘣—吧 嘣 嘣—吧"，韵脚多落强拍。速率高于 trap | 叙事、自陈、复古、稳。情绪平实有力 | 柔情向叙事可借 |
| **Trap flow** | 慢主拍（70–90 BPM 主拍）+ 密集 hi-hat + 重 808。三连音节拍是标志 | 当代主流、戏谑、燃、城市夜 | 快嘴向默认 |
| **三连音 flow（triplet / Versace flow）** | 把一拍三等分，每字占 1/3 拍。**和中文天然契合**——中文词组常是三字（双截棍 / 龙卷风 / 菊花台） | 洗脑、推进、卡点。抖音爆款常用 | 两个子变体都可点缀 |
| **Double-time flow（快嘴 / 快三连音）** | 半拍内塞 3 个字，听感骤快。提速 + 炫技 | 高潮段、控诉、情绪爆发 | 快嘴向高潮段 |
| **切分 flow（syncopation）** | 重音故意偏离常规强拍，制造律动错位 | 增加变化，避免顺口溜 | 任意段落做变化用 |
| **反拍 / 掉拍（layback）** | 故意比 beat 慢半步唱，松弛感 | 慵懒、自陈、稳。难度较高 | 柔情向自陈段 |

### Flow 编排的两条铁律
- **韵脚不要全落强拍**：4/4 拍里把韵脚全压在军鼓位（强拍），flow 会一段一段、像顺口溜。把部分韵脚 / 重音排到弱拍或切分点上，flow 才"流动"。
- **停顿要出乎意料**：一个 8 拍里什么时候停很关键。停顿太规律 → 顺口溜；太没规律 → 失控。停顿是 flow 设计的一部分，墨九写词时就要标出来（用换行 / 空格 / 标点）。

## 三、押韵体系（重点：消灭"把单押误报成双押"）

中文押韵看的是**韵母（韵腹+韵尾）的发音**，不是声母，也不是字面。声调标准上要求一致，但演唱可微调（见下文）。

### 按"句尾连续押韵字数"分类

| 类型 | 定义 | 判定 | 难度 |
|---|---|---|---|
| **单押** | 句尾**最后 1 个字**韵母相同 | 只看末尾单字 | 低（流行歌默认） |
| **双押** | 句尾**连续 2 个字**的韵母**都**相同（声调标准上也要一致） | 末尾两字逐字比韵母 | 中 |
| **三押** | 句尾连续 3 个字韵母都相同 | 末尾三字逐字比 | 高 |
| **多押 / N 押** | 句尾连续 4 字及以上 | 同理逐字比 | 很高，且需 flow 配合才听得出 |

#### 双押到底是什么——给正反例（防误报）

**双押 = 末尾两个字的韵母逐字连韵，不是只有最后一个字押。**

✅ 正例（真双押）：
- 「新闻 xīn wén」/「惊人 jīng rén」—— 倒数第二字 in/ing 韵区近、最后字 en 韵相同，两字都连
- 「爆炸 bào zhà」/「叫骂 jiào mà」—— ao + a，两字都连
- 「世界 shì jiè」/「致谢 zhì xiè」—— i + ie，两字都连
- 「新房客」/「金黄色」—— 这是六字韵脚里的三字段，每字逐位连韵

❌ 反例（这些是单押，不要报成双押）：
- 「我的家 jiā」/「一朵花 huā」—— 只有最后一个字 a 押，倒数第二字「的 de」vs「朵 duǒ」不押 → **单押**
- 「天黑了 le」/「风停了 le」—— 最后字「了」同字不算押韵技巧，倒数第二字「黑 hēi」vs「停 tíng」不押 → **单押**（甚至是同字偷懒）
- 「想回家 jiā」/「在天涯 yá」—— 末字 a 押，倒数第二字「回 huí」vs「天 tiān」不押 → **单押**

**判定动作**：拿两个句尾，从最后一个字往前**逐字比韵母**。连续相同的字数 = 押的字数。第 2 个字一断 → 就是单押，不许标双押。

**别凭脑子现凑韵脚** → 查 `knowledge/rap-rhyme-bank.md`（rap 韵库）。韵库按十三辙分区，每区有常用收字表 + 预攒的现成双押词组（每条已标连韵两字 + 声调），直接挑用，不卡壳也不误报。库里没有再现造，造完按上面判定法验证并回填韵库。

### 按韵脚位置 / 变化分类

| 类型 | 定义 | 判定要点 |
|---|---|---|
| **内韵（内部押韵 / 夹韵）** | 押韵点出现在**句子中间**，不在句尾 | 押韵点是否在非句尾位置？是则为内韵。可与单/双押叠加 |
| **隔行押（ABAB）** | 隔一句押一次（第 1、3 句押，第 2、4 句押另一韵） | 看押韵句的间隔 |
| **转韵 / 换韵** | 整首中途换韵脚 | 段与段之间韵脚变了。中文说唱**鼓励换韵**——一韵到底易审美疲劳 |
| **抢韵 / 跨行押** | 韵脚提前到下一句开头，或押韵跨越行界 | 韵脚不落在预期的行末，制造连贯推进感 |

### 押韵的三条纪律（直接对治"为韵害意"）
1. **押韵服务内容，不是反过来**。GAI 的写法是范本：不狂热堆韵，更在意内容、思想、节奏，很少出现滞涩的险韵和尴尬的俗韵。胡乱四押、丢了逻辑和主题的套词儿，只会让歌变质。
2. **多押要 flow 配合**。写出多押但 flow 不把那几个字唱在节奏同一位置，听众根本听不出来——等于白写。墨九写多押时要在创作笔记里注明"这几字需同位重音"，提示阿声。
3. **声调**：标准双押要求声调一致，演唱时可把不同声调唱成相同（vocal delivery 覆盖字本身声调）。但 Suno 不会主动这么做——**给 Suno 的歌词，押韵优先选声调也一致的**，否则韵会被旋律抹平（见第五节）。

## 四、Punchline 构造法

Punchline = 歌词的炸点，让人"哦！"的那一句。不是押韵爆点，是恰到好处的文字运用。

### 核心结构：Setup → Punch
前面几句作 **setup**（铺垫语境 / 营造预期），最后一句作 **punch**（反转 / 揭示）。也可连续 punch 打组合拳。

### 写出真正炸点的四种手法

| 手法 | 做法 | 例 / 说明 |
|---|---|---|
| **谐音双关** | 用发音相同/近、意义不同的词制造双层意思 | AR「烦人的 faker」把人名和 faker 叠一起。**关键：要让人"秒懂"**——get 不到就尴尬 |
| **一词多义 / 语义双关** | 利用同一个词的"字面义 + 圈内义/引申义" | 一个词同时挂两层意思（如「绿」字面色彩 + 引申义）。指桑说槐，曲折含蓄 |
| **意象反转** | setup 铺一个意象的常规联想，punch 句把它翻面 | 把熟悉意象推到反预期方向 |
| **概念碰撞** | 把两个不相干领域的概念并置，撞出新意 | 避免「AA 像 BB」直白比喻——换个奇思妙想的方式说 |

### 双关 punchline 成立的三要素
写一句双关，缺一不可：
1. **双重语境**：一句话能同时被放进两个理解框架（字面 / 引申）
2. **触机（trigger）**：让听众恰好在某点联想到第二层的触发因素
3. **语言铰链（hinge）**：连接两层意思的那个词本身（谐音词或多义词）

### Punchline 纪律
- **从字面到引申都要通顺**——两层意思单独看都成立。
- **依赖背景知识的 punch 要确认听众有那个知识**，否则只有圈内人 get 到。
- punchline 可写进韵脚（人名 / 多义词放句尾强拍），但不是必须。
- **抖音场景**：rap 段的 punchline 就是金句式 hook 的来源，墨九写时标出"这句是 punchline / 金句候选"，给抖叔小汽切片用。

## 五、中文说唱特有的坑

中文说唱有英文说唱没有的硬约束，外加 Suno 生成的已知问题。墨九阿声都要扫。

### 1. 声调与 flow 打架
中文每个字带声调，rap 的 flow 有自己的旋律 / 节奏走向，两者天然冲突。化解：
- 成熟做法是演唱时把字的声调"唱"成 flow 需要的样子（vocal delivery 覆盖声调）。
- **但 Suno 不会主动这么做**——它倾向按字的标准声调念。所以给 Suno 的词，押韵尽量选声调也一致的韵脚，flow 走向尽量顺着声调，别和声调对着干。

### 2. 咬字密度的矛盾
密度越高（一拍塞越多字）越炫技，但听感清晰度越差。中文每行建议 **10–15 字**，信息密度别拉满。连音（三连音 / 五连音 / 七连音）是炫技手段，**加太多 flow 不稳、字也糊**——点到为止。

### 3. 生僻字 / 多音字
说唱信息密度高，多音字生僻字风险比抒情歌更大。处理沿用 `knowledge/suno-vocal.md` 的双版本歌词策略（发布展示版用汉字 + Suno 输入版做拼音替换）。

### 4. Suno 生成 rap 的已知问题
- **押韵被旋律化抹平**：Suno 容易把 rap 段唱得偏旋律，韵脚的"押"被旋律盖掉。对策：押声调也一致的韵 + Style 里明确 `rap` / `spoken delivery` 标签。
- **double-time 糊字**：快段塞字多，Suno 易吐字不清。对策：double-time 段更要严控发音——避免连续卷舌音（zh/ch/sh）和连续爆破音（b/p/d/t）相邻；该段字数比常规段再压一点。
- **跳词 / 念错**：生僻多音字 Suno 会跳过或念错 → 拼音替换。
- **自检法**：写完自己念一遍歌词，**念到哪卡壳哪就有问题**——自己念不顺，Suno 多半也唱不好。
- **局部修，别整首重抽**：只差一点点不对，用 Re-run / Replace Section 针对出错片段小修，比反复抽卡省积分。
- **Style 正向描述**：`Clear Pronunciation` / `Clean Delivery` / `Tight Phrasing` 提清晰度。**负面提示词陷阱**：别在 Style 里写 "No XXX"，Suno 看到 XXX 反而会加；V5 用 Advanced Options 的 Exclude。

## 六、Benchmark 拆解（学习优秀中文说唱）

跑 `chinese-rap` 新歌前，墨九 / 阿声按题材找对应类型的 benchmark 借手法。下面每首拆到可借鉴的具体手法，不是泛泛点评。

### 类型 A：叙事说唱

**《垃圾话》— GAI**
- **flow**：boom-bap 取向，规整律动承载叙事，不炫快
- **押韵**：不堆韵，避开险韵俗韵；押韵让位于叙事逻辑
- **结构亮点**：**戏剧化多视角叙事 + 首尾呼应**。20 字内两个角色四句对话，结尾把男主从白日梦拉回现实。情绪曲线"由黑暗到光明再被现实拉回"
- **可借**：rap 不一定要快要炸——叙事体 rap 靠视角切换和结构设计取胜。柔情向自传题材直接参考这个结构思路

**《听妈妈的话》— 周杰伦**
- **flow**：R&B 基底 + 半说半唱（spoken-sung），钢琴分解和弦做回忆感铺垫
- **押韵**：**词曲咬合极好，中文四声与旋律走向高度一致**——这正是对治"声调与 flow 打架"的范本
- **结构亮点**：**"别人 / 我"场景对比叙事**（"别人在玩游戏 / 我却靠在墙壁背我的 ABC"）；完整情感曲线：童年不解→青年领悟→成年感恩，靠"说唱→歌唱"的音乐形式本身表现认知转变
- **意象**：用日常意象承载抽象情感（温暖的食谱 / 美丽的白发）；幽默自嘲消解说教感
- **可借**：柔情向 rap 的黄金模板——具体生活细节对比 + Verse rap / Chorus melodic 切换 + 自嘲口吻去严肃

### 类型 B：抖音爆款说唱

**《八方来财》/《大展鸿图》— 揽佬**
- **flow / beat**：**孟菲斯说唱（Memphis rap，Trap 分支）**——粗粝、low-fi 采样，鼓机 + 合成器，**重复的副歌旋律**。中速 trap 区间（主拍约 70–85 BPM，hi-hat 双倍 / 三连音填充）
- **结构亮点**：**采样做钩子**——《大展鸿图》前奏采样粤剧《帝女花》，孟菲斯碎拍鼓机撞粤剧锣鼓，"摇又不太摇"的张力；副歌"来财"是极简洗脑段
- **为什么火**：强节奏卡点的抓耳 hook + 贴大众的情绪价值（"发财"心理）+ 方言说唱新鲜感
- **可借**：爆款 rap 的 hook 要短到能洗脑（2–3 字反复）；采样一段熟悉的旋律做记忆钩子；人声是主武器、采样是副武器，采样可"略带失真脏感"

**《跳楼机》— LBI 利比**
- **flow / 风格**：嘻哈灵魂乐（hip-hop soul）= **情歌旋律 + 快速饶舌**，点缀转音
- **结构亮点**：主歌（钢琴铺寂寥）→ 预副歌（加鼓 + 贝司提节奏）→ 副歌（情感释放）三段递进；"4536251 万能和弦"；编曲细节服务歌词（合成器倒放音效=时光倒流，失真音色模拟电梯下坠）
- **可借**：rap 不必全程念白——"情歌旋律 + 快饶舌"对柔情向极友好；副歌前置进开头 30 秒；旋律走向"向上走"，哪怕歌词 emo 旋律也别太悲

### 类型 C：国风 / 神话题材说唱

**《哪吒》— GAI（新华流）**
- **flow / 风格**：新华流——以说唱为本，以民乐赋彩；战斗题材偏快偏硬，可推到 85–110 BPM，高潮段可上 double-time
- **编曲**：hip-hop beat 上叠传统民乐——锣鼓提气势、唢呐 / 竹笛做战斗色彩，常叠 808 / 电子。羽调式（小调五声）偏好
- **押韵 / 词**：用小学 3000 常用字写宏大叙事（天 / 山 / 灯 / 家 / 门 / 路），寥寥几笔写大意象；规避谐音歧义
- **可借**：国风 rap 的配方 = hip-hop 鼓 + 民乐点缀（古筝替吉他、二胡替弦乐）；神话题材契合 rap 的"反叛 / 突破"内核（哪吒抗命）；意象用常用字、降理解门槛

**《飞升》/《山海神话》系 — 龚琳娜**
- **结构亮点**：多 rapper 各演一个神（女娲 / 二郎神 / 孙悟空 / 哪吒），各用自己的声线和技法塑造角色
- **可借**：神话题材 rap 可走"一段一个角色 / 视角"的结构；高难度演唱技法对位特定神兽形象

## 七、写 rap 的最小动作清单

**墨九（写词）**：
1. 读 brief 画像字段 → 定 `chinese-rap` 子变体（柔情 / 快嘴）
2. 按子变体定 flow 类型（柔情向 boom-bap / 慢三连音；快嘴向 trap / double-time）
3. 先排 bar 和停顿，再填字——每行字数别忽长忽短
4. 押韵：查 `rap-rhyme-bank.md` 按情绪选韵区、挑现成双押词组；选服务内容的韵；逐字判定单/双押别误报；多押标注"需同位重音"
5. 至少设计 1 个 setup→punch 的 punchline，标出金句候选
6. g2p 扫雷（多音字 + 低频字）→ 出 Suno 输入版
7. 自己念一遍，卡壳处改
8. 收尾：把本首新用到、自检/试唱确认押得顺的双押词组 / 收字回填 `rap-rhyme-bank.md` 对应韵区的 🧪 节

**阿声（写 prompt）**：
1. 读 brief 画像 + 墨九的 flow 标注
2. 按 flow 类型选 beat 风格（boom-bap / trap / 孟菲斯）和 BPM
3. Style 里明确 `rap` / `spoken delivery` 防旋律抹平押韵
4. 加 `Clear Pronunciation` / `Tight Phrasing`；double-time 段标清晰度
5. 负面约束走 Advanced Options Exclude，别在 Style 写 "No XXX"
